# cffga.github.io

Página personal. Disponible en [cffga.github.io](https://cffga.github.io).

#Contenido

- **Piezas sueltas** — textos donde expongo mi punto de vista sobre temas matemáticos y afines.
- **Programación** — materiales de programación en español, incluyendo un curso introductorio de Python.
- **Proyectos** — proyectos en matemáticas, estadística y ciencia de datos.
- **Notas** — apuntes de trabajo y textos en progreso.
- **Material recomendado** — libros, videos y recursos que considero valiosos.

#Tecnología

Sitio estático en HTML/CSS puro, sin frameworks. El curso de Python está construido con [Quarto](https://quarto.org).
